# Orbital Resonance

A physics-based puzzle game created for the JS13K competition. Navigate through gravitational fields to collect targets while avoiding planetary collisions.

## Game Features

- **Realistic orbital mechanics**: Experience authentic gravitational physics simulation
- **Procedural generation**: Each level features unique planet configurations
- **Progressive difficulty**: Increasing complexity with more planets and targets per level
- **Dual control scheme**: Use keyboard (WASD/arrows) or mouse for intuitive ship control
- **Visual feedback**: Particle effects for thrust, explosions, and target collection
- **Minimalist aesthetic**: Clean, space-themed visuals optimized for performance

## Controls

- **W/↑/Space**: Thrust forward
- **A/←**: Rotate left  
- **D/→**: Rotate right
- **Mouse**: Aim ship toward cursor
- **Click/Hold**: Thrust
- **R**: Restart after game over

## Objective

Collect all green targets in each level while avoiding collision with red planets. Use gravitational forces strategically to navigate through space efficiently.

## Technical Details

- **Size**: ~5.7KB (well under 13KB js13k limit)
- **Technology**: Pure HTML5 Canvas with JavaScript
- **Physics**: Custom gravitational simulation with velocity integration
- **Optimization**: Heavily minified code maintaining full functionality
